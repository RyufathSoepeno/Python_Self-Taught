# Colons

Strict Equality (===)
This is like == only difference is it make sure the data type is similar, not just the result (2.5 can be float and double)

Tip: (:) is for inside the bracket, (=) is for assign/store

ðŸ§  (=) vs (:) 

(:) can be used for

Range-based for loops: In a range-based for loop (C++11 and later), the colon separates the iteration variable from the range to be iterated over.
Example: for (auto x : range) {}.

Conditional (Ternary) Operator: As part of the ? : operator.
Example: auto r = b ? "yes" : "no";.

Member Initializer Lists: In a constructor definition, a colon introduces a member initializer list to initialize member variables and call base class constructors before the constructor's body executes. This is more efficient than assignment within the body and necessary for const members and references.
Example: MyClass() : member(value) {}.

Inheritance: In a class or struct declaration, the colon specifies the base class(es) from which the class inherits.
Example: struct Derived : Base {}.

Access Specifiers: Used after access specifiers (public:, private:, protected:) within a class declaration.

Bit Fields: In a struct or class, a colon can define the number of bits a data member should occupy, allowing memory packing.
Example: struct S { bool x : 1; };.
